# 🚀 Déploiement Ultra Rapide (5 minutes)

## ⚡ **Solution la Plus Simple : Vercel**

Vercel c'est encore **plus facile que Netlify** !

### **3 Clics pour Déployer :**

#### **Clic 1️⃣ : Allez ici**
```
https://vercel.com/new
```

#### **Clic 2️⃣ : Connectez-vous**
- Cliquez **"Continue with GitHub"**
- Si pas de GitHub, créez un compte gratuit : https://github.com/signup
- Autorisez Vercel

#### **Clic 3️⃣ : Importez votre projet**
1. Cliquez **"Import Git Repository"**
2. Collez cette URL :
```
https://github.com/votre-username/pcgi87-beneficiaire-pwa
```
*(À condition que vous ayez mis le code sur GitHub)*

**OU** utilisez **Drag & Drop** :

```bash
cd C:\Users\Wonde\pcgi87\pcgi-beneficiaire-pwa
npm run build
```

Puis glissez le dossier `dist` ici : https://vercel.com/new
