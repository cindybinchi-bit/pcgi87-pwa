# 📱 Installation "Mon Espace" sur Mobile

## 🚀 Option 1 : Installation PWA (Recommandée - Gratuit & Facile)

La PWA fonctionne comme une **app native** sur tous les smartphones.

### ✅ Sur iPhone (iOS)

1. Ouvrez **Safari**
2. Allez sur le lien fourni par votre référent
3. Appuyez sur le bouton **Partage** (flèche vers le haut)
4. Sélectionnez **"Sur l'écran d'accueil"**
5. Confirmez → L'app s'installe automatiquement ! 🎉

### ✅ Sur Android

1. Ouvrez **Chrome** (ou votre navigateur)
2. Allez sur le lien fourni par votre référent
3. Appuyez sur le **menu (⋮)** en haut à droite
4. Sélectionnez **"Installer l'application"** ou **"Ajouter à l'écran d'accueil"**
5. Confirmez → L'app s'installe automatiquement ! 🎉

---

## 📲 Fonctionnalités de l'App

Une fois installée, vous avez :
- ✅ Accès hors-ligne (données en cache)
- ✅ Icône sur l'écran d'accueil comme une vraie app
- ✅ Notifications (bientôt)
- ✅ Mise à jour automatique
- ✅ Aucun téléchargement nécessaire (utilise le navigateur)

## 🆘 Dépannage

### "L'option ne s'affiche pas"
- ✓ Vérifiez que vous utilisez **Safari (iOS)** ou **Chrome (Android)**
- ✓ La page doit être en **HTTPS** (pas HTTP)
- ✓ Attendez le chargement complet

### "L'app se ferme au lancement"
- ✓ Videz le cache du navigateur
- ✓ Réinstallez en suivant les étapes ci-dessus

### "Je ne vois pas l'agenda de mon activité"
- ✓ Vérifiez que votre référent a bien assigné votre bénéficiaire
- ✓ Glissez vers le bas pour actualiser
- ✓ Déconnectez-vous et reconnectez-vous
